# Stories Companion - MHS3

A local-first companion app for Monster Hunter Stories 3 players. Look up Head-to-Head attack patterns and monster weaknesses on the fly, and build your own living encyclopedia as you explore.

## Tech Stack

React 19 | TypeScript | Vite | Tailwind CSS | Zustand | React Hook Form

## Getting Started

```bash
yarn install
yarn dev
```

## Scripts

| Command      | Description              |
| ------------ | ------------------------ |
| `yarn dev`   | Start dev server         |
| `yarn build` | Type-check and build     |
| `yarn lint`  | Run ESLint               |
